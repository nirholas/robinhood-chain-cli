Quickstart

Node ≥ 20. Reads work with zero setup; writes need a wallet.

# install globally
npm install -g hood-cli

# instant read — no config, no wallet
hood price AAPL

# the full Stock Token board, live
hood stocks --sort premium --dex

# stream new memecoin launches
hood launches --follow

# set up a wallet (encrypted keystore, password-protected)
hood config set wallet

# quote a swap (default: dry run)
hood swap --sell USDG --buy WETH --amount 100

# sign and send — always confirms first
hood swap --sell USDG --buy WETH --amount 100 --execute

Every command

Reads need nothing. Writes need a wallet + --execute — quote/plan is always the default. Full reference with live --help output: commands.html.

hood price <symbol>

Chainlink oracle price, live DEX price, and the premium between them.

hood stocks

The full 95-token Stock Token board, sortable by symbol, price, or premium.

hood coins

Memecoin screener — newest launches or ranked by bonding-curve activity.

hood launches --follow

Live-streaming feed of new NOXA and The Odyssey launches.

hood portfolio <address>

Multiplier-correct positions and USD totals for any wallet.

hood tx / hood token

Decoded transaction inspection and token metadata via Blockscout + RPC.

hood watch <addr|token>

Live activity stream — transfers and native ETH movement.

hood swap

Quote by default; --execute signs and sends a real Uniswap v3 swap.

hood transfer

Send ETH or any ERC-20, with an always-on confirmation gate.

hood deploy-token

Deploy a real, fixed-supply ERC-20 from a JSON config.

hood faucet

Testnet faucet instructions + live testnet balance check.

hood config

RPC, wallet, network, and spend-cap settings.

Guard rails

Every write defaults to a dry run. Nothing signs without an explicit gate.

--execute required

Swaps, transfers, and deploys print a plan by default. Nothing is sent until you pass --execute.

Interactive confirm

Even with --execute, a yes/no prompt renders the recipient, amount, and network before signing — skip it only with --yes.

Stock Token eligibility

Stock Tokens are tokenized securities. Acquiring one requires --acknowledge-eligibility — an explicit operator affirmation, never assumed.

Spend cap

hood config set maxSpendUsd <n> refuses any swap or transfer whose estimated USD value exceeds your cap.

Encrypted keystore

Wallet keys are AES-256-GCM encrypted under a scrypt-derived password — never plaintext on disk.

Meaningful exit codes

Scriptable: 2 = bad usage, 3 = network, 4 = guard refused, 5 = not found, 6 = wallet needed.
